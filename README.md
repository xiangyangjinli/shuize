### 🌊 水泽 (ShuiZe)

> 水善利万物而不争，处众人之所恶，故几于道。

**水泽** 是一款高隐蔽性的内网资产测绘与渗透框架 / 轻量级分布式数据采集引擎。正如其名，它致力于像水一样汇聚信息、渗透壁垒，在静默中完成复杂的安全评估任务。

### 🌊 项目简介

在复杂的网络环境中，水泽 旨在解决目标用户痛点，如：传统工具特征明显、数据孤岛严重、配置繁琐的问题。通过 核心技术点，如：模块化设计 / 动态流量伪装 / 智能算法，它能够为红队行动/数据开发提供源源不断的“水源”支持。

### ✨ 核心特性

- **🌊 汇聚 (Aggregation)**
  强大的资产/数据整合能力。支持http，tcp，能够像百川归海一样，快速构建目标的全景视图。
- **💧 渗透 (Infiltration)**
  极致的隐蔽与兼容性。采用去特征化流量技术，能够平滑穿越各类 防火墙/EDR/网络限制，润物细无声。
- **🌫️ 幻化 (Metamorphosis)**
  高度灵活的插件系统。用户可根据实际场景自由编排工作流，支持 自定义脚本/热更新，随形就势，应对万变。
- **⚡ 奔涌 (Performance)**
  基于 Golang 打造，具备极高的并发处理能力和极低的资源占用。
